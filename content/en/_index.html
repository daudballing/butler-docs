{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} Butler includes a palette of features: - A REST API making it easy to access Sense features from 3rd party systems, or from within the load script. API endpoints include starting tasks, sending Slack messages and publishing data to MQTT messages. - Instructions and sample files for hooking into Sense's logging framework and get notified when tasks fail, users log on and more. - Use MQTT to start Sense reload tasks. {{% /blocks/lead %}} {{< blocks/section color="dark" >}} {{% blocks/feature icon="fa-lightbulb" title="Latest release: 3.0" %}} Enable exactly the REST API endpoints you need - no more, no less.
